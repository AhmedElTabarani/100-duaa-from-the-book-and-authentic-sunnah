# 100 دعاء من الكتاب والسنة الصحيحة

كتاب [100 دعاء من الكتاب والسنة الصحيحة](./100%20دعاء%20من%20الكتاب%20والسنة%20الصحيحة%20-%20محمد%20صالح%20المنجد%20بعد%20تنقيح%20الخميس.pdf) هو كتاب يحتوي على 100 دعاء من الكتاب والسنة الصحيحة، وقد تم تجميعها بواسطة الشيخ محمد صالح المنجد

المشكلة هي أنك كمبرمج لا تستطيع الاستفادة من الكتاب لأنه `PDF` ولا توجد نسخة `JSON` متوفره لها  
لذا قررت تحويل أدعية الكتاب وتنظيمها في ملف `JSON` لتسهيل الاستفادة منها ووضعت معها المصادر اين ذكر هذا الدعاء في القرآن والأحاديث النبوية

# هيكل ملف الـ JSON

```json
[
  {
    "id": "ترتيب الدعاء في الكتاب",
    "category": "تصنيف الدعاء",
    "duaa": [
      {
        "text": "نص الدعاء",
        "source": {
          "type": "quran أو hadith",
          "references": [
            {
              "surah": {
                "number": "رقم السورة",
                "name": "اسم السورة"
              },
              "ayah": {
                "from": "بداية الآية",
                "to": "نهاية الآية"
              }
            }
          ]
        },
        "vocabulary": [
          {
            "text": "النص",
            "meaning": "معنى النص"
          }
        ]
      }
    ]
  }
]
```

### مثال عندما يكون المصدر حديثًا نبويًا

```json
{
  "text": "اللَّهُمَّ إِنِّي أَسْأَلُكَ الْجَنَّةَ، وَأَعُوذُ بِكَ مِنَ النَّارِ",
  "source": {
    "type": "hadith",
    "references": [
      {
        "rawi": "راوي الحديث",
        "mohdith": "من حدث بالحديث",
        "book": "الكتاب الذي ذكر فيه الحديث",
        "numberOrPage": "رقم الحديث أو الصفحة",
        "grade": "حكم الحديث",
        "takhrij": "أين تم تخريج نفس الحديث في الكتب الأخرى"
      }
    ]
  },
  "vocabulary": null
}
```

## ملحوظات

- العنصر `duaa` هي مصفوفة من الأدعية،  قد تجد دعاءً أو دعاءين فقط، لأن الشيخ أحيانًا يضع دعاءين في نفس الترتيب، والحد الأقصى دعاءان
- مصدر الدعاء `source.type` يكون إما `"quran"` أو `"hadith"`
- العنصر `source.references` يحتوي على مراجع المصدر:
  - إذا كان `type` هو `"quran"`: يحتوي على `surah` و`ayah`
  - إذا كان `type` هو `"hadith"`: يحتوي على `rawi` و`mohdith` و`book` و`numberOrPage` و`grade` و`takhrij`
- العنصر `vocabulary` يحتوي على شرح معاني بعض النصوص الواردة في الأحاديث فقط
  - يكون `null` إذا لم تكن هناك مفردات تحتاج إلى شرح
  - يكون دائمًا `null` عندما يكون مصدر الدعاء من القرآن
- قد يكون هناك بعض الأخطاء في تشكيل بعض الأحاديث
- الأحاديث مختارة من موقع [الدرر السنية](https://dorar.net/)
  - حاولت أن أجد اقرب الأحاديث الصحيحة التي تحتوي عن نفس كلمات الدعاء
  - رتبت الأحاديث حسب الأقرب للدعاء أو أحيانا بحسب المحدث
